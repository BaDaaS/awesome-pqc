# Awesome (Practical) Post-Quantum Cryptography

This document aims to gather information about practical post-quantum cryptography.
It does not aim to be an exhaustive research document about post-quantum
cryptography; the goal is to only focus on what is actually useful for the next
5-10 years.

For instance, the document won't cover hypothetical physical foundations of
quantum computers, but would try to focus on the physics used by companies
building actually quantum computers. It doesn't mean that alternatives are not
worth exploring; we only target currently or near-term usable technologies.

## Introduction

A quantum computer is an alternative to classical computer.
In quantum computing, we use terms as
[qubits](https://en.wikipedia.org/wiki/Qubit), [quantum
registers](https://en.wikipedia.org/wiki/Quantum_register) (an "aggregation" of
qubits", as "classical" registers are an "aggregation" of bits).
